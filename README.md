# 概要
デザインパターンの検討

# インストール
- `git clone https://github.com/KamijoKeiko/design-pattern.git`


# 使い方
- `cd design-pattern`
- `python main.py`

# ファイル構成
- main.py : メイン処理
- state_pattern.py : ステートパターン
- command_pattern.py : コマンドパターン
- mail_sender.py : メール送信処理クラスのイメージ
- request_json.py : フロントエンドからのリクエストを模倣した変数を定義
